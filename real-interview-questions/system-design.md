## 🔹 Architecture & HLD

32. **How would you build a real-time dashboard displaying live data updates?**
    - _Also asked as:_ WebSockets vs Polling.

33. **How do you design frontend for offline support?**
    - _Also asked as:_ Service workers and offline-first concepts.

34. **How would you build a high-performance UI that renders massive tables with live updates?**

35. **Discuss the HLD of your current or past frontend system or project.**
